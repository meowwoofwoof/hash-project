\doxysubsection{Detailed Description}
Консольное приложение для вычисления и проверки хешей файлов. 

Поддерживаются алгоритмы MD5, SHA‑1, SHA‑256. Реализация всех хеш‑функций написана вручную. Пользователь может выбрать режим\+: получение хеша или проверка соответствия хеша.
