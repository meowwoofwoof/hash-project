◆ verify_md5()

bool verify_md5 ( const std::string & path,
const std::string & expected )

Проверяет совпадение хеша файла с ожидаемым значением.

Функция вычисляет хеш указанным алгоритмом и сравнивает результат с эталонной строкой, заданной пользователем.

Parameters
filepathПуть к проверяемому файлу.
expectedОжидаемый хеш в нижнем регистре (hex).
algoСтрока‑идентификатор алгоритма: "md5", "sha1" или "sha256".
Returns
true — если вычисленный хеш совпадает c expected;
false — в противном случае или при неизвестном алгоритме.
